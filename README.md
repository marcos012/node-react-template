# Node React template
<img src="https://i.imgur.com/Xa2uYZM.png" /><br>
Application template with ReactJS && Node.js by: [Marcos Paulo](https://github.com/marcos012)

## Installation: <br>

- [Node/NPM](https://nodejs.org/en/) <br>


**Backend:**

```shell
$ cd server
$ npm install
$ npm run db:migrate
$ npm run dev
```

**Frontend:**

```shell
$ cd web
$ npm install
$ npm start
```
