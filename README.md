## Installation: <br>

- [Node/NPM](https://nodejs.org/en/) <br>


**Backend:**

```shell
$ cd server
$ npm install
$ npm run db:migrate
$ npm run dev
```

**Frontend:**

```shell
$ cd web
$ npm install
$ npm start
```
